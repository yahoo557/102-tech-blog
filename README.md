
# 블로그 서비스 프로젝트 1.0

학습내용과 프로젝트 진행 내용을 정리하기 위해 제작한 블로그 서비스입니다.


## Tech Stack

**Client:** Vue.js  @3.2.45

**Server:** Nest.js @9.2.0
 
**DB:** PostgreSQL @14.5

**Infra:** EC2, S3, RDS

**Util:** SwaggerUI for api Docs
